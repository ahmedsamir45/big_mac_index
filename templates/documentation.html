API Documentation

Overview

This API provides data visualization for the Big Mac Index, allowing users to explore price variations across countries and time periods.

The application is designed to help users compare the cost of a Big Mac in different currencies, track historical price trends, and visualize economic disparities globally.

Features

  • Interactive global map visualization of the Big Mac Index.
  • Historical data tracking for multiple countries.
  • Time-series analysis of Big Mac prices.
  • Comparative bar charts for different regions.
  • Scatter plots showing the relationship between GDP and Big Mac prices.

Available Endpoints

1. Home

Endpoint: /

Method: GET

Returns the main page with the visualization interface, including map and data selection options.

2. Get Data

Endpoint: /data

Method: GET

Returns Big Mac Index data for the selected date, including country-specific pricing information.

Query Parameters:

  • date (optional) - Specify a date to filter the data. If omitted, the latest available data will be used.

Response: JSON object containing map data, country prices, and available dates.

3. Country Details

Endpoint: /country/<country_code>

Method: GET

Returns detailed historical pricing data for a given country, allowing for in-depth economic analysis.

Path Parameters:

  • country_code - The ISO A3 country code (e.g., USA, GBR, CHN).

Response: JSON list of historical price records for the specified country, including currency exchange rates and local prices.

4. Charts Page

Endpoint: /charts

Method: GET

Returns the charts page with additional visualizations, including bar charts, time-series plots, and scatter diagrams.

5. Chart Data

Endpoint: /chart_data

Method: GET

Returns data for various visualizations, including comparative price analysis and economic insights.

Response: JSON object containing:

  • Bar chart data (top 15 most expensive countries based on Big Mac price).
  • Time-series data for key economies (e.g., USA, UK, China, Japan, Germany).
  • Scatter plot data showing the relationship between GDP and Big Mac prices.

6. Documentation Page

Endpoint: /documentation

Method: GET

Returns this documentation page, providing details on API usage and available features.
